프로젝트 개요
 - 프로젝트 목적: 객체지향 프로그래밍(OOP)의 핵심 원칙을 적용하여 확장 가능한 커머스 시스템의 기초 구조 설계

기술 스택
 - Language: Java 23
 - IDE: IntelliJ IDEA
 
주요 기능
 - 카테고리 기반 상품 조회
 - 장바구니 관리
 
패키지 구조
src/
├── commerce/
│   ├── Main.java
│   ├── Category.java
│   └── CommerceSystem.java
│   └── Customer.java
│   └── Product.java
│   └── productCart.java
└── README.md
 
실행 방법
 - Main 클래스 main 메서드 실행
 - 화면에 출려된 카테고리 번호를 선택하여 상품목록 집입
 - 상품 상세 정보 확인 후 장바구니 담기 또는 취소
 - 장바구니에 상품이 들어가면 장바구니 메뉴 활성화
 - 장바구니 메뉴를 통한 주문 확정 또는 장바구니 담기 취소
